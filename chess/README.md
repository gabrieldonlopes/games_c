
# ♟️ chess

[![C](https://img.shields.io/badge/C-98.6%25-00599C.svg)]()

Xadrez completo e funcional com interface gráfica e lógica de debug.

##  Headers utilizadas
- SDL.h
- SDL_Image.h
- SDL_ttf.h

##  Funcionalidades
- Tabuleiro 8x8 representado com caracteres
- Movimentação de peças (peão, torre, cavalo, bispo, rainha, rei)
- Controle de turnos (brancas vs pretas)
- Detecção de xeque e xeque-mate (parcial)
- Movimentações especiais (castling, promotion, en passant)

### Compilar e Executar
```bash
cd games_c/chess
cmake --build build 
build/chess 
```
